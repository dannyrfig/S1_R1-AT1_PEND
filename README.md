# Rápido Seguro Logística

Projeto desenvolvido com HTML5, CSS3 e Bootstrap 5, com foco em layout responsivo e estilização de uma página inicial com formulário de login.

## Descrição

A página contém:

- Menu de navegação superior
- Logo centralizada e responsiva
- Formulário de login com validação
- Rodapé com links
- Estilização personalizada com CSS

## Tecnologias Utilizadas

- HTML5  
- CSS3  
- Bootstrap 5.3  
- Media Queries  

## Estrutura

### Header
Utiliza Flexbox e classes `nav` e `nav-pills` do Bootstrap para criar o menu de navegação. Possui cor de fundo personalizada no CSS.

### Logo
Centralizada com `display: block` e `margin: auto`.  
Responsiva com `max-width` e ajustes via Media Queries:
- A partir de 400px: 50%
- A partir de 700px: 30%

### Formulário
Contém:
- Campo de e-mail com `required` e `pattern`
- Campo de senha obrigatório
- Botão estilizado com `btn btn-primary`

Centralizado com classes Flex do Bootstrap.

### Footer
Links organizados com `nav` do Bootstrap e texto centralizado.
